## Features

- Community feed with posts
- Threaded nested comments
- Like system with karma
- Top contributors leaderboard (last 24 hours)
- Responsive design

---

## How to Run Locally

```bash
git clone https://github.com/Abhivamsh/community-frontend.git
cd community-frontend

npm install
npm run dev
```

---

## Environment Variables

Create a `.env` file:

```
VITE_API_URL=http://localhost:8000/api
```

For production, this points to the deployed backend.
